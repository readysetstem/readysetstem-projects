Randomize

Often when creating games, you'll need to add a random element. For example, imagine sports video game, where you are playing against the computer. If the opposing team ran the same exact plays every time you played the game, the game would get pretty boring.

This is very true in the Simon Says projects game, where the computer must play a random sequence for the player to repeat. If the sequence isn't random, it's not nearly as interesting of a game (although it still would get quite hard as the sequence got longer).

Python provides a set of functions in the random module that can return random numbers. We'll describe below how a few of them work...

One of the simplest functions is randrange(stop). It returns a random number greater than or equal to 0, but less than stop. For example, if you try:

You might get:

0
3
3
2

But if you run it again, you'll almost certainly get a different set of numbers.

Another convenient function is randint(a, b). It returns a random number between a and b (inclusive). For example, randint(5, 8) can return 5, 6, 7, or 8.

kiwi

But again, if you run it again, the output will surely be different!
