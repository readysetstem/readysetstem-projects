In this project, we will allow our Minecraft character to place a bomb anywhere in the Minecraft world with a the press of a button, and then detonate that bomb with another press of the button.

In our game loop, there are two modes:

  • In place mode, when the button is pressed, we place a bomb at the current location in from of Steve. Then, we change to detonate mode.

  • In detonate mode, when the button is pressed, we detonate the bomb that was previously placed (using a separate detonate() function). Then, we change to place mode.

To keep track of the modes, we use the variable place_mode, which will be True when we're in place mode. The game starts in place mode.

To find out where the bomb is placed, we use the pollBlockHits() function, similar to the way we used it in the project.

Finally, out detonate() function is identical to the dig() function in the project.
