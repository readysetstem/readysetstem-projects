Now, let's take a look at what our code is doing:

  • In Lines 5 through 15, we create the Speech() that we'll play throughout the program. We'll also set up a list, actions_tts, that is the speech for each action that a user can take. The list is in the same order as the variables NOACTION, LEFT, etc, which is important. We'll use this list to let the user know which action we've detected.

  • In Line 15, we let the user know the game is starting.

  • In Line 32, we request the user to perform an specific tilt. Of course, in this version of the game, the tilt is still always just LEFT! (Don't worry, we'll change it next project).

  • In Lines 47 through 53, instead of just printing out if the right or wrong action was peformed, we speak to the user. And, it was wrong, we tell them the action that we think they did.

  • In Line 58, we let the user know the game has ended.
