In this project, we integrate the multiple buttons we implemented several projects ago along with the note/sound generation from the sound functions and the speaker.

Using the hardware setup from , we can turn our CREATOR Kit into a very basic piano. We will start with four buttons each wired to a GPIO as input, and write some code that will generate corresponding note when each button is pressed. We'll keep it simple in this project, and just play each note for one second.

Try this, and you'll likely notice that the there's a popping/crackling sound while you hold the button down. The reason is that when the button is pressed, the program will try to replay the note over and over again, each time through the while loop. When you replay a note (that is, play a note that is already playing), you'll hear a little pause as the note stops and restarts. That is the clicking you hear — and we'll fix this in the next project.
