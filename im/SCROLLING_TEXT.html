In this project, we will draw some text and scroll it across the screen.

Your breadboard should still have the circuit we wired in the project. In addition, we've added a button that we'll use to control what we display on the LED matrix. As a reminder, here is what the breadboard should look like:

Give it a try. You should now see the intro text scroll across the screen. Press the button, and the text should switch.

  1. Can you reverse the text? It will now only be readable with a mirror.

  2. The first time the button is pressed, the text switches direction. Can you have another button press make the text switch back?

  3. Can you make a additional button presses cause the text to scroll faster with each successive press?

  4. In our project, the vertical position doesn't change. Change that so the text bounces — as it moves across the screen, alternate the y position up and down one point.
