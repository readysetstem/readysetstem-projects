Making Notes

Songs are made up of sequences of notes (or combinations of multiple notes). The Ready Set STEM API provides a Note() function to create notes of different pitches.

The basic use of the Note() function is to create a note with:

Note(pitch)

where pitch can be either a number which represents the frequency of the pitch, or a letter A though G which is the name of the note.
