LED MATRIX

Introduction to the LED Matrix and libraries

    • First step, power on (2 wires) - should see vertical and horizontal bars sweep across display

    • Second step, hook it up (4 wires)

    • Optional: mounting posts

    • Quick test - is it connected.  If this program runs without error, then the LED Matrix is connected.

from rstem.led_matrix import FrameBuffer

FrameBuffer()

    • A framebuffer is where the next image to be displayed on the screen is stored.  General usage:

      • erase()

      • draw stuff

      • show()

      • ... repeat

    • New code - a dot on the screen.

from rstem.led_matrix import FrameBuffer



fb = FrameBuffer()



x, y = (3, 3)

fb.erase()

fb.point(x, y)

fb.show()
