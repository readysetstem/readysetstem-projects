The is_pressed() function is simple and easy to use, but has a couple issues. One issue is that if our code isn't checking very frequently if the button is pressed, it can miss a press. For example, say we write some code that checks once a second to see if the button is pressed. If a user pushes and releases the button quickly (say in a quarter of a second), it is likely the code will miss the button press — it might check before and after the button is pressed, but not check during the press.

The Button() object fixes this problem with a couple new functions. The function presses() returns the number of times the button has been pressed since the last time the function was called. Similarly, the function releases() returns the number of times the button has been released since the last time the function was called.
